# Lumpy-HMM: Bayesian Hidden Markov Model with Tweedie-GAM Emissions

A PyMC-based implementation of a **Bayesian Hidden Markov Model** for **lumpy transactional data** (extreme zeros + heavy-tailed spend) using **Tweedie-GAM emissions** and **Sequential Monte Carlo (SMC)** inference.
